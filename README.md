# 少女前线抽卡记录工具

一个用于获取和管理《少女前线2追放》抽卡记录的桌面应用。

## 📖 使用教程

[点击观看详细视频教程](https://www.bilibili.com/video/BV1oaz2BsEPo/)

## 🚀 快速开始

### 1. 安装证书

首次使用需要安装 mitmproxy 证书：

- 点击左侧的「安装证书」按钮
- 在打开的网页中，选择对应的操作系统（Windows/macOS/Linux）
- 下载并按照页面指引安装证书
- **重要**：安装后将证书设置为"受信任的根证书颁发机构"

### 2. 获取抽卡记录

- 安装证书后，点击「更新记录」按钮
- 打开游戏并打开抽卡界面
- 记录会自动获取并显示在界面上
- 支持三个池子：角色池、武器池、常驻池

### 3. 导入/导出记录

**导出记录：**
1. 在输入框中设置一个 ID（例如：`player1`）
2. 点击「导出记录」按钮
3. 记录会保存在程序目录的 `records` 文件夹中

**导入记录：**
1. 在输入框中输入之前导出的 ID
2. 点击「导入记录」按钮
3. 记录会加载到当前界面

### 4. 检查更新

- 点击「检查更新」按钮可查看是否有新版本
- 如有更新，可跳转到 GitHub 下载

## ⚠️ 注意事项

1. **程序单实例运行**：不要重复打开程序，否则可能导致代理冲突

2. **证书安装**：
   - 证书必须安装到"受信任的根证书颁发机构"

3. **网络要求**：
   - 需要保持网络连接

4. **记录管理**：
   - 导出记录会合并已存在的同 ID 记录
   - 建议定期导出备份抽卡记录

## 📁 文件说明

- `records/` - 存放导出的抽卡记录

## 🔧 常见问题

**Q: 点击「更新记录」没有反应？**
A: 请检查证书是否正确安装，或尝试重启程序。

**Q: 提示获取记录失败？**
A: 确保游戏已打开抽卡界面，并保持网络连接。

**Q: 重复打开程序会怎样？**
A: 程序会提示已运行并自动退出，这是正常行为。
